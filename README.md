# HAM Maze v2

HAM Maze is a daily web3 maze game. This version has been updated to run on **Stacks** using **Clarity** smart contracts. 

## Structure

- `/ham-web`: The Next.js web application frontend.
- `/stacks-contracts`: The Stacks Clarity smart contracts and Clarinet configuration.

## Setup

1. **Smart Contracts**: Navigate to `/stacks-contracts` and use Clarinet or deploy via the Hiro Platform.
2. **Web App**: Navigate to `/ham-web`, run `npm install`, and then `npm run dev` to start the local development server.
